🌐 Responsible AI in Global Data-Driven Decision Making

🎓 Capstone Project | Ethical AI & Explainability
An interactive diagnostic platform designed to bridge the gap between complex Machine Learning models and ethical accountability. This tool allows stakeholders in Finance, Healthcare, and HR to audit datasets for bias and understand AI "black-box" decisions through Local Interpretable Model-agnostic Explanations (LIME).

## 🚀 Key Features
Global Domain Context:Tailored regulatory insights based on the selected industry (e.g., GDPR, EU AI Act).
Automated Bias Detection: Calculates Disparate Impact Ratios to identify unfairness toward sensitive protected groups (Gender, Race, Age).
XAI (Explainable AI): Integration of LIME to provide row-level transparency on why the model made a specific prediction.
Feature Importance: High-level visualization of the top drivers behind global model decisions.
Transparency Report: Generates a downloadable summary of fairness metrics and performance for institutional accountability.

## 🛠️ Tech Stack
**Frontend/Backend:** [Streamlit](https://streamlit.io/)
**Machine Learning:** Scikit-Learn (Logistic Regression, Decision Trees, Random Forest)
**Explainability:** LIME (Local Interpretable Model-agnostic Explanations)
**Data Handling:** Pandas & NumPy
**Visualizations:** Matplotlib

💻 How to Run
1. Clone the repository:
   git clone [https://github.com/your-username/Responsible-AI-Global.git](https://github.com/your-username/Responsible-AI-Global.git)
   cd Responsible-AI-Global

2. Install dependencies:
   pip install -r requirements.txt

3. Launch the application:
   streamlit run app.py
   
⚖️ Ethical Standards Applied
This project follows the **80% Rule (Four-Fifths Rule)** for disparate impact and aligns with the **OECD AI Principles** for transparency and robust decision-making.
